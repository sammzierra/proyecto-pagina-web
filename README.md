#  TechStore - E-commerce de Práctica

Este es un proyecto de una tienda virtual interactiva desarrollado como parte de mi formación en el tecnólogo **ADSO (SENA)**. El objetivo es demostrar habilidades en el desarrollo Frontend y la manipulación del DOM con JavaScript.

##  Tecnologías utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Diseño responsivo con Flexbox y CSS Grid.
* **JavaScript (Vanilla):** Lógica del carrito de compras y renderizado dinámico de productos.

##  Características
* Diseño moderno con bordes redondeados y sombras suaves (estilo Pro).
* Carrito de compras funcional (agregar/eliminar productos).
* Contador de productos en tiempo real.
* Modal interactivo para visualizar el detalle de la compra.

##  Cómo ejecutarlo
1. Clona el repositorio.
2. Abre el archivo `index.html` en tu navegador.
